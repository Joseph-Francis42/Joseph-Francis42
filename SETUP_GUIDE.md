2. Enter exact repository name:

3. Select **Public**.

## 📦 Step 2: Upload Files to your Repository

Upload all the files from this directory to `Joseph-Francis42/Joseph-Francis42`:

### Push via Git Terminal:
```bash
git init
git add README.md banner.png .github/workflows/snake.yml
git commit -m "Initialize animated GitHub profile"
git branch -M main
git remote add origin https://github.com/Joseph-Francis42/Joseph-Francis42.git
git push -u origin main
```

## 🐍 Step 3: Trigger the Snake Contribution Animation

1. On your GitHub profile repository (`Joseph-Francis42/Joseph-Francis42`), click the **Actions** tab.
2. Select **Generate Snake Contribution Animation** on the left menu.
3. Click **Run workflow** -> **Run workflow**.
4. Once completed, your profile README will display the animated snake eating your contribution grid!
